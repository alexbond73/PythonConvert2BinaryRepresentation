# PythonConvert2BinaryRepresentation
An elegant converter of integer to a binary value, represented as Python list

Examples of use:
$ python3 int2bin.py 3
[1, 1]

$ python3 int2bin.py 4
[1, 0, 0]

$ python3 int2bin.py 255
[1, 1, 1, 1, 1, 1, 1, 1]

$ python3 int2bin.py 256
[1, 0, 0, 0, 0, 0, 0, 0, 0]
